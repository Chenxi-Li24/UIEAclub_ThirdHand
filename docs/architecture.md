## 模块职责

| 模块 | 位置 | 芯片 | 职责 |
|------|------|------|------|
| 外骨骼 | `firmware/s3-exo/` | ESP32-S3 | 传感器采集、UI显示、机器人控制中转 |
| 终端 | `firmware/p4-terminal/` | ESP32-P4 (规划) | 大屏交互、摄像头、语音 I/O、LLM Agent |
| 可视化 | `visualizer/` | PC/浏览器 | Three.js 3D 机械臂模型 |
| AI Agent | `voice-ai/` | PC | LLM tool-calling 循环 |

## 通信协议

### 法奥机械臂

| 方向 | 端口 | 协议 | 格式 | 用途 |
|------|------|------|------|------|
| ESP32 → 法奥 | 20007 | UDP | `/f/bIII...III/b/f` | 伺服控制帧 |
| 法奥 → ESP32 | 20005 | TCP | CNDE 二进制 | 实时状态（关节角/速度/扭矩） |
| PC → ESP32 | 20008 | UDP | 文本命令 | `status` / `servo j1 ...` |

### 内部通信（规划）

| 方向 | 协议 | 用途 |
|------|------|------|
| S3 ↔ P4 | WiFi UDP | 传感器数据上行、指令下行 |
| P4 ↔ PC | WiFi HTTP | LLM Agent 调用 |

## 外骨骼引脚分配

| 引脚 | 用途 | 总线 |
|------|------|------|
| 6,7,5,4,8,15 | ST7789V SPI 屏幕 | SPI |
| 11,10 | I2C (IMU + 触摸) | I2C |
| 13,14 | CST816T 触摸 INT/RST | GPIO |
| 21 | WS2812 LED | NeoPixel |
| 42 | PWM 蜂鸣器 | LEDC |
| 1 | 电池 ADC | ADC |
| 0 | BOOT 按钮 | GPIO |
| 16,17,18 | I2S 麦克风（规划） | I2S |
| 2 | I2S 功放（规划） | I2S |
